# Google Stitch 디자인 산출물

효자손 UI는 Google Stitch MCP로 초안을 생성하고, `public/`에 자체 CSS로 재구현했습니다.
(Stitch 원본 HTML은 CDN Tailwind 의존이라 배포용으로 부적합 → 동일 디자인 토큰으로 재작성)

- Stitch 프로젝트: `projects/14280820096904212343` (효자손 — 서류 읽어주는 AI 손주)
- 디자인 시스템: `assets/10126541537274495677` (노인친화: NOTO_SANS, 메인 #1B6BD6, 신호등 컬러 예약)
- 화면: 홈 `eab33a...`, 결과 `ac554a...`, 도움말 `08bab7...` (도움말은 스크린샷만 제공됨)

| 파일 | 내용 |
|---|---|
| `stitch_홈.png/html` | 홈 화면 디자인 |
| `stitch_결과.png/html` | 분석 결과 화면 디자인 |
| `stitch_도움말.png` | 도움말 화면 디자인 |

지원서 'UX·디자인' 항목의 근거 자료로 활용 가능.
