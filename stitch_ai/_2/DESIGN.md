# 효자손 디자인 시스템 (어르신 전용 웹앱)

## 원칙
- 사용자는 65세 이상 디지털 취약 고령층. 모든 요소는 초대형·고대비·단순.
- 기본 본문 글자 20px 이상, 버튼 텍스트 24px 이상, 페이지 제목 32px 이상. 절대 16px 미만 금지.
- 터치 타깃 최소 56px 높이. 핵심 CTA 버튼은 화면 폭 전체 + 높이 96px 이상.
- 색: 메인 파랑(#1B6BD6, 신뢰). 신호등 시맨틱 컬러는 안전 #1E8E3E(초록)/주의 #F9AB00(노랑)/위험 #D93025(빨강)로 예약 — 메인 UI에 남용 금지.
- 배경 흰색(#FFFFFF), 본문 텍스트 #1A1A1A. 명도 대비 WCAG AAA 지향.
- 둥근 모서리 16~24px, 부드럽고 친근한 카드형 레이아웃.
- 애니메이션 최소화. 장식 아이콘보다 큰 이모지+한글 레이블 병기.
- 말투: 존댓말, 초등 5학년 어휘. 영어/전문용어 금지 (예: OK→확인).
- 하단 고정 액션바 사용, 버튼에는 아이콘+텍스트 병기.
